Overview#

The apstools package provides supplemental support for data acquisition using the Bluesky framework at the Advanced Photon Source. The support consists of a thin Python interface to EPICS record structures and databases not already supported in ophyd. Additional support is provided for certain types of scans and data file writer support for common formats (such as SPEC and NeXus). See headings in the Summary Contents below for more information:
