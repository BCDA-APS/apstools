apstools#

Library of Python tools for use with Bluesky at the APS.

The apstools package provides supplemental support for data acquisition using the Bluesky framework at the Advanced Photon Source. The support consists of a thin Python interface to EPICS hardware, records, databases, and other structures not already supported in ophyd.
